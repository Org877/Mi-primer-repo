Mi aplicación

¿Qué es FlutterFlow?

**FlutterFlow** es una plataforma visual que permite crear aplicaciones móviles sin necesidad de escribir mucho código. Utiliza **Flutter** como el framework de Google, para crear aplicaciones nativas para Android, iOS y web. Ademas de que permite a los usuarios diseñar interfaces de usuario mediante un sistema de arrastrar y soltar, acelerando el desarrollo de aplicaciones. Además, permite la integración fácil con **Firebase** para gestionar bases de datos, autenticación y más.

¿Qué es Firebase?

**Firebase** es una plataforma de desarrollo de aplicaciones que ofrece una variedad de servicios como bases de datos en tiempo real, autenticación de usuarios, almacenamiento de archivos y notificaciones push.

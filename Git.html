Qué es un Repositorio
Qué es la Llave SSH
Un repositorio es un espacio donde se almacena, organiza y comparte información. Además, puede ser utilizado para guardar información y tener un control de versiones.
Las llaves SSH se crean para un repositorio con el fin de establecer una conexión segura entre la computadora y el servidor, que en este caso es GitHub. Estas llaves permiten realizar operaciones como `push`, `pull` y `clone`.
Qué es Git
Cómo se sube todo
Git es un sistema de control de versiones creado por Linus Torvalds en 2005.
Para subir todo a un repositorio, se deben usar comandos como `git add .`, `git commit -m "cambios"`, y `git push`.
Comandos de Git
Imágenes
Comandos básicos:
(Git init: inicia un nuevo repositorio)
(Git clone: Hace una copia del repositorio remoto a la PC)
(Git add: Agrega un archivo)
(Git commit -m "algo": Guarda los cambios y los sube en el historial del repositorio)
(Git push: Permite enviar los cambios realizados al servidor)
(Git status: Muestra el estado del directorio en el que estas trabajando)
(Git pull: Descarga un repositorio remoto e inmediatamente lo fusiona en el repositorio actual. )
